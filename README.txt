Installation:
1. Go to chrome://extensions
2. Enable Developer Mode in the top right
3. Select "Load Unpacked" in the top left
4. Choose the Tetrome folder (the folder that this txt file is inside)
5. It may not be pinned, so open the Puzzle Piece icon in the extensions list to access it (you can then pin it).
